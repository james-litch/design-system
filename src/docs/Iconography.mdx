# Iconography

This package contains the icons used across the design system. Its important to use these icons to ensure consistency across the design system.

For React projects, icons can be imported directly from :

```jsx
import { IconCheck } from '@scandotcom/design-system/icons'

<IconCheck />
```

For Rails projects icons can be added via the inline_svg gem. To learn more about the inline_svg gem please [click here](https://github.com/jamesmartin/inline_svg)

```erb

<%= inline_svg('check.svg') %>

```
