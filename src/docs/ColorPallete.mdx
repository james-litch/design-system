# Color Palette

Here are the colors used across the design system.

## Usage 

The colors displayed can be used by applying tailwind style tokens. 

```HTML
<div class="bg-pink-300"></div>
```

With tailwind v4 we can use the css color variables like so:

```css
color: var(--color-black);
background-color: var(--color-pink-300);
```
